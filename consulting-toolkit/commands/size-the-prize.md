description: Estimate the financial impact of an opportunity or initiative
---
Size the financial impact of: $ARGUMENTS

1. Define the value drivers (revenue uplift, cost reduction, risk mitigation)
2. For each driver:
   - Estimate the magnitude (show your math)
   - Assign a confidence level
   - Note key assumptions
3. Aggregate into total addressable value
4. Apply a realization factor (what % can realistically be captured)
5. Present in a one-page format suitable for a CFO audience
