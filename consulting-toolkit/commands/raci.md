Create a RACI matrix for: $ARGUMENTS

1. Identify key activities/decisions
2. Identify key stakeholders/roles
3. Assign R (Responsible), A (Accountable), C (Consulted), I (Informed)
4. Validate: exactly one A per row, at least one R per row
5. Present as a clean table
6. Note any gaps or potential conflicts
