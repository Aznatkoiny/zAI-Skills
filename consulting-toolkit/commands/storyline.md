---
description: Convert analysis into a pyramid-principle slide storyline
---
Create a slide storyline for: $ARGUMENTS

1. Identify the governing thought (the #1 message)
2. Define 3-4 supporting pillars
3. For each pillar, define the evidence slides needed
4. Write action titles for EVERY slide (complete sentences that
   convey insight, not description)
5. Note the visualization type for each slide (chart, table, text, diagram)
6. Test: reading only the titles should tell the complete story

Output as a numbered storyline document.
