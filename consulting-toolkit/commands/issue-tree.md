Decompose this problem into a MECE issue tree: $ARGUMENTS

1. State the core question clearly
2. Break into 3-5 mutually exclusive, collectively exhaustive branches
3. Go 2-3 levels deep on each branch
4. At each leaf, note what analysis would answer that question
5. Present as an indented tree structure
6. Highlight the 2-3 branches most likely to drive the answer
