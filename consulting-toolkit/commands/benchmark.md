Run a benchmarking analysis for: $ARGUMENTS

1. Define the benchmarking scope:
   - What metrics/practices are we comparing?
   - Who are the relevant peers/comparators?
2. Gather data for each comparator:
   - Key performance metrics
   - Operational practices
   - Strategic positioning
3. Normalize for comparability (adjust for size, geography, mix)
4. Identify performance gaps:
   - Where does the subject outperform?
   - Where does it underperform?
   - What drives the gaps?
5. Extract actionable insights:
   - Best practices to adopt
   - Quick wins vs. structural changes
   - Estimated impact of closing gaps
6. Present as a structured comparison table with narrative

Use web search for real data. Source all data points.
